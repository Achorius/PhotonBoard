1. Introduction

1.1 Présentation

PhotonBoard est un logiciel professionnel de commande d'éclairage DMX512 conçu pour le spectacle vivant, l'architecture lumineuse, les events et les installations scéniques. Il fonctionne sur macOS, Windows et Linux.

Multi-univers DMX

Jusqu'à 16 univers DMX simultanés, avec sortie ArtNet et USB DMX.

Bibliothèque de 20 000+ fixtures

127 fixtures intégrées, accès en ligne à l'Open Fixture Library, import GDTF, QLC+, Avolites, GrandMA, CSV.

Moteur d'effets LFO

9 formes d'ondes, mode spatial 3D, effets composés multi-canaux.

Timeline DAW

Éditeur temporel avec clips, marqueurs, zones, support timecode.

Visualisation 3D

Prévisualisation en temps réel des faisceaux, positions et couleurs.

Contrôle MIDI complet

MIDI Learn sur tous les contrôles, support encodeurs et faders.

Mode poursuite

Suivi de cible avec manette de jeu, convergence automatique des faisceaux.

Éditeur de fixtures

Créez vos propres définitions pour les projecteurs chinois, LED bars, etc.

1.3 Installation

macOS

Téléchargez le fichier .dmg depuis la page de téléchargement.
Double-cliquez sur le .dmg et glissez PhotonBoard dans le dossier Applications.
Au premier lancement, macOS peut bloquer l'application. Allez dans Réglages Système → Confidentialité et sécurité, puis cliquez « Ouvrir quand même ».

Windows

Lancez l'installeur .exe téléchargé.
Suivez les étapes de l'assistant d'installation.
PhotonBoard se trouve ensuite dans le menu Démarrer.

Linux

Téléchargez le fichier .AppImage ou .deb.
Pour AppImage : rendez le fichier exécutable (chmod +x PhotonBoard.AppImage) puis lancez-le.

2. Prise en main rapide

2.1 Premier lancement

Au premier lancement, PhotonBoard crée automatiquement un nouveau show vide. Le dossier de sauvegarde par défaut est :

  • macOS / Linux : ~/Documents/PhotonBoard/
  • Windows : C:\Users\[Nom]\Documents\PhotonBoard\

Les fichiers show utilisent l'extension .pbshow (format JSON).

Si un show précédent a été sauvegardé, PhotonBoard le recharge automatiquement au démarrage.

3. Patch — Adressage DMX

L'onglet Patch permet de configurer l'adressage DMX de chaque projecteur, gérer la bibliothèque de fixtures et organiser les projecteurs en groupes.

3.1 Ajouter des projecteurs

Cliquez sur le bouton Add Fixture en haut du panneau Patch.
Dans la fenêtre modale, parcourez la bibliothèque locale (onglet « Local Library ») ou recherchez en ligne (onglet « Online (OFL) »).
Sélectionnez le fabricant puis le modèle de projecteur souhaité.
Choisissez le mode DMX (ex. « 16 Channel », « 22 Channel »).
Définissez l'univers et l'adresse de départ.
Indiquez la quantité si vous ajoutez plusieurs projecteurs identiques (les adresses s'incrémentent automatiquement).
Cliquez Add pour ajouter au patch.
Les adresses DMX se calculent automatiquement : si un projecteur en mode 16 canaux commence à l'adresse 1, le suivant commencera à l'adresse 17.

3.2 Bibliothèque de fixtures

PhotonBoard inclut 127 définitions intégrées de projecteurs professionnels couvrant les principales marques :

Generics

Dimmer, RGB, RGBW, RGBWAU, Moving Head Wash/Spot/Beam, LED Bar, Fog Machine, Hazer, Strobe, Sunstrip, Fresnel, Blinder, Wall Washer

Marques professionnelles

ADJ, Astera, Ayrton, Cameo, Chauvet DJ/Pro, Clay Paky, Elation, ETC, GLP, High End Systems, Martin, Robe, Robert Juliat, SGM, Vari-Lite

Marques événementielles

BriteQ, Eurolite, Fuzzix, Showtec, Stairville, TMB

Fixtures personnalisées

Créez vos propres définitions via l'éditeur intégré pour les projecteurs non référencés

3.3 Recherche en ligne (OFL)

L'onglet Online (OFL) dans la fenêtre d'ajout permet de rechercher parmi plus de 20 000 fixtures de l'Open Fixture Library.

Cliquez l'onglet « Online (OFL) » dans la modale d'ajout.
Tapez le nom du fabricant ou du modèle dans le champ de recherche (ex. « Robe Robin »).
La recherche s'effectue automatiquement après 400 ms. Les résultats affichent le fabricant, le modèle et les catégories.
Cliquez Download à côté du modèle souhaité. La fixture est téléchargée, convertie au format natif et ajoutée à votre bibliothèque locale.
La recherche en ligne nécessite une connexion Internet. Pour un usage hors-ligne, pré-téléchargez vos fixtures ou importez-les depuis une clé USB (voir section 3.4).

3.4 Importer des fichiers fixtures

3.5 Créer / éditer une fixture

Pour les projecteurs non référencés (LED chinoises, matériel custom…), PhotonBoard intègre un éditeur de fixtures complet.

Cliquez Create Fixture dans l'onglet Patch.
Renseignez le nom, le fabricant et la catégorie.
Ajoutez les canaux DMX un par un, ou utilisez les préréglages rapides : Dimmer, RGB, RGBW, Pan/Tilt, Gobo, Shutter.
Pour chaque canal, définissez le type, la valeur par défaut et la priorité (HTP ou LTP).
Configurez les modes en activant/désactivant les canaux par mode (certains projecteurs ont plusieurs modes : 6ch, 12ch, 16ch…).
(Optionnel) Remplissez les propriétés physiques : dimensions, poids, puissance, angle de faisceau.
Cliquez Save. La fixture est ajoutée à la bibliothèque locale.
Les fixtures créées par l'utilisateur sont stockées dans le dossier de données de l'application et peuvent être éditées ou supprimées. Les fixtures intégrées ne peuvent pas être supprimées.

3.6 Groupes

Les groupes permettent de sélectionner et contrôler rapidement un ensemble de projecteurs.

  • Créer un groupe — Depuis l'onglet Patch ou Faders, créez un groupe et attribuez-lui une couleur.
  • Ajouter des fixtures — Glissez des fixtures dans un groupe ou utilisez le menu d'édition du patch.
  • Sélection rapide — Cliquez sur un groupe dans la barre latérale pour sélectionner toutes ses fixtures.
  • Utilisation — Les groupes sont utilisables dans les Faders, Effects, Follow et la ligne de commande.

4. Faders — Commande manuelle

4.1 Vue d'ensemble

L'onglet Faders offre un contrôle direct de chaque canal DMX via des faders verticaux, similaire aux consoles physiques.

Organisation

  • Sélecteur d'univers — Menu déroulant pour basculer entre les univers (0–15).
  • Sélecteur de groupe — Filtrage rapide par groupe ou « All » pour tout afficher.
  • Affichage par fixture — Chaque fixture affiche son nom, son mode et ses canaux individuels.

Code couleur des faders

CouleurType de canal
OrangeIntensity / Dimmer
CyanPan / Tilt (position)
Rouge / Vert / BleuCanaux couleur (RGB)
GrisCanaux génériques

Interaction

  • Glisser un fader pour changer la valeur (0–255).
  • Clic droit sur un fader → menu contextuel avec MIDI Learn.
  • La valeur numérique est affichée sous chaque fader.

4.2 Le Programmer

Le Programmer est une couche temporaire qui capture les modifications manuelles. C'est le cœur du workflow d'enregistrement :

  1. Montez les faders ou utilisez le picker couleur → les valeurs entrent dans le Programmer.
  2. Le Programmer a la priorité la plus haute dans le mixage DMX (au-dessus des cuelists et effets).
  3. Record : enregistre le contenu du Programmer dans une mémoire (cue).
  4. Clear : vide le Programmer (les valeurs reviennent aux niveaux des playbacks actifs).
Règles de mixage : les canaux d'intensité (Dimmer) suivent la règle HTP (Highest Takes Precedence — la valeur la plus haute gagne). Tous les autres canaux suivent la règle LTP (Latest Takes Precedence — la dernière action gagne).

5. Fixtures — Contrôle individuel

5.1 Sélection de fixtures

Le panneau gauche liste tous les projecteurs patchés. Cliquez pour sélectionner, Shift+clic pour une sélection multiple. Utilisez le sélecteur de groupe pour filtrer.

5.2 Picker couleur

Pour les fixtures disposant de canaux couleur (RGB, RGBW, CMY…), un picker couleur graphique apparaît :

  • Gradient 2D — Cliquez pour choisir la teinte et la saturation.
  • Curseurs R, G, B — Ajustement fin de chaque composante (0–255).
  • Curseur Dimmer — Intensité globale de la fixture.

5.3 Pad XY (Pan/Tilt)

Pour les lyres (moving heads), un pad XY de type joystick permet de contrôler la position :

  • Axe horizontal — Contrôle du Pan (panoramique).
  • Axe vertical — Contrôle du Tilt (inclinaison).
  • Glissez le point pour positionner le faisceau.
  • Les valeurs en degrés sont affichées en temps réel.
Le mode Sheet (feuille de calcul) affiche toutes les fixtures et tous leurs canaux dans une grille éditable, pratique pour modifier plusieurs valeurs simultanément.

6. Playback — Scènes et Cuelists

Le système de Playback permet de créer, enregistrer et restituer des scènes lumineuses.

6.1 Créer une cuelist

Depuis l'onglet Playback, cliquez + New Cuelist.
Nommez la cuelist (ex. « Opening », « Verse 1 », « Finale ») et attribuez une couleur.
La cuelist apparaît dans la liste avec ses contrôles : GO, Stop, Back, Fader, Flash.

6.2 Enregistrer des mémoires (cues)

Créez votre « look » en montant les faders, couleurs, positions dans le Programmer.
Sélectionnez la cuelist cible.
Cliquez Record — le contenu du Programmer est capturé comme nouvelle cue (mémoire).

Paramètres de chaque cue

ParamètreDescription
NomLibellé de la mémoire
Fade InTemps de montée en secondes (0 = instantané)
Fade OutTemps de descente en secondes
DelayDélai avant le début du fondu
FollowTemps en secondes avant auto-avance à la cue suivante (vide = attente du GO manuel)

6.3 Restitution et fondus

BoutonAction
GOLance la cue suivante (avec les fondus définis)
BackRevient à la cue précédente
StopArrête la cuelist (relâche les valeurs)
FaderMaster de la cuelist (0–100 %)
FlashPlein feux tant que le bouton est maintenu

Le tracking est actif par défaut : les valeurs persistent d'une cue à l'autre sauf si elles sont explicitement modifiées.

6.4 Chases

Les chases sont des séquences automatiques qui bouclent entre plusieurs pas.

  • BPM — Vitesse de défilement en battements par minute.
  • Direction — Forward, Backward, Bounce, Random.
  • Fondus — Temps de fondu entre chaque pas.
  • Tap Tempo — Tapez le tempo en cliquant ou via MIDI.

7. Effects — Moteur d'effets LFO

Le moteur d'effets génère des oscillations automatiques sur les canaux DMX. Il est idéal pour les mouvements de lyres, les pulsations de couleur et les effets de balayage.

7.1 Créer un effet

Depuis l'onglet Effects, cliquez + New Effect ou choisissez un modèle prédéfini (Dimmer Pulse, Pan Sweep, Color Pulse…).
Sélectionnez les types de canaux affectés (Dimmer, Pan, Tilt, Red, Green, Blue…).
Réglez les paramètres (vitesse, amplitude, forme d'onde, phase, fan).
Cliquez Play pour activer l'effet.

7.2 Formes d'ondes

7.3 Paramètres d'effet

7.4 Mode spatial

Le mode spatial utilise les positions 3D des projecteurs pour propager les effets de manière réaliste :

  • Index — Phase répartie selon l'ordre de patch (par défaut).
  • X (gauche–droite) — Propagation horizontale.
  • Y (haut–bas) — Propagation verticale.
  • Z (avant–arrière) — Propagation en profondeur.
  • Radial — Propagation depuis le centre de la scène.
Les effets peuvent être enregistrés dans les cues : lors de l'enregistrement d'une mémoire, les effets actifs sont capturés et relancés automatiquement lors de la restitution.

8. MIDI — Contrôle externe

PhotonBoard supporte le contrôle MIDI complet via l'API Web MIDI, compatible avec tous les contrôleurs USB et Bluetooth MIDI.

8.1 Connexion d'un contrôleur

Branchez votre contrôleur MIDI USB (ou activez le Bluetooth MIDI).
Ouvrez l'onglet MIDI. Le panneau gauche affiche la liste des appareils détectés avec un indicateur vert (en ligne) ou rouge (hors ligne).
Le moniteur MIDI affiche en temps réel le dernier message reçu (type, canal, numéro CC, valeur).

8.2 MIDI Learn

L'assignation MIDI se fait de deux manières :

Méthode 1 — MIDI Learn global

Cliquez le bouton MIDI Learn dans la barre d'outils.
Cliquez sur le contrôle à assigner dans PhotonBoard (fader, bouton, etc.).
Bougez le contrôle MIDI physique (fader, bouton, encodeur).
L'assignation est créée automatiquement. Cliquez à nouveau « MIDI Learn » pour quitter le mode.

Méthode 2 — Clic droit

Faites un clic droit sur n'importe quel fader ou bouton → sélectionnez MIDI Learn dans le menu contextuel → bougez le contrôle MIDI.

9. Stage Layout — Éditeur 2D

L'éditeur de plan de scène offre une vue de dessus interactive pour positionner les projecteurs.

Fonctionnalités

  • Glisser-déposer — Déplacez les fixtures sur le plan pour refléter leur position réelle.
  • Rotation — Utilisez la poignée de rotation pour orienter les projecteurs.
  • Sélection multiple — Shift+clic pour sélectionner plusieurs fixtures.
  • Grille — Taille configurable (0.5–5 m), accrochage magnétique activable.
  • Zoom et panoramique — Molette pour zoomer, clic-glisser sur le fond pour panoramiquer.
  • Image de fond — Importez un plan de salle ou une photo aérienne.

Affichage des fixtures

Chaque fixture est représentée par un rectangle coloré selon son groupe, avec :

  • Le nom du projecteur
  • L'adresse DMX
  • Un indicateur de direction (angle de visée)

Dimensions de la salle

Configurez les dimensions dans le panneau latéral : Largeur, Profondeur et Hauteur en mètres. Ces dimensions sont partagées avec le Visualizer 3D.

Le zoom et le panoramique sont conservés lorsque vous changez d'onglet et revenez au Stage Layout.

11. Follow — Mode poursuite

Le mode Follow permet de faire converger les faisceaux de plusieurs lyres sur un point mobile de la scène, contrôlé par une manette de jeu (gamepad).

Configuration

  • Sélection des fixtures — Choisissez les lyres (moving heads) qui participent à la poursuite.
  • Sensibilité — Ajustez la vitesse de déplacement de la cible.
  • Hauteur de la cible — Hauteur par défaut (1.5 m = tête d'une personne debout).
  • Inversions — Inversez les axes X, Y ou Z selon votre configuration.

Commandes gamepad

ContrôleAction
Stick gaucheDéplacement de la cible gauche/droite et avant/arrière
Stick droit / GâchettesRéglage de la hauteur de la cible
Bouton d'activationActive/désactive le suivi

L'état du gamepad est affiché dans le panneau : nom, connexion, niveau de batterie et boutons actifs.

Le système calcule automatiquement les angles Pan/Tilt de chaque lyre par cinématique inverse, en tenant compte de la position 3D et de l'orientation de montage de chaque projecteur.

12. Live — Timeline DAW

L'éditeur Live offre une timeline de type DAW (station de travail audio numérique) pour orchestrer un show complet de manière séquentielle et synchronisée.

12.1 Pistes et clips

  • Pistes multiples — Ajoutez autant de pistes que nécessaire pour organiser vos éléments.
  • Clips — Chaque clip représente une cuelist placée à un instant précis. Glissez un clip pour le déplacer dans le temps.
  • Durée — Ajustez la durée de chaque clip. Les fondus d'entrée et de sortie sont visibles graphiquement.
  • Couleurs — Chaque clip peut être coloré pour une meilleure lisibilité.

12.2 Marqueurs et zones

Marqueurs

Les marqueurs sont des repères temporels nommés et colorés (jaune, orange, rouge, vert, bleu, violet). Cliquez sur un marqueur dans la règle pour y naviguer instantanément.

Zones

Les zones définissent des sections nommées de la timeline (ex. « Intro », « Verse 1 », « Chorus », « Finale »). Elles sont réordonnables et colorées pour une navigation visuelle rapide.

12.4 Timecode

Le mode timecode permet de synchroniser la timeline avec une source externe :

  • Formats — MM:SS ou HH:MM:SS:FF
  • Cadences — 25, 29.97, 30, 48, 50, 59.94, 60 fps
  • Activation — Toggle « Timecode » dans le panneau de contrôle
Les contrôles Play/Stop de la timeline sont assignables en MIDI pour un déclenchement depuis une console physique.

13. Settings — Paramètres

13.1 Sorties ArtNet

ArtNet est le protocole standard pour transporter le DMX512 sur réseau Ethernet (Ethernet → nœud ArtNet → câble DMX → projecteurs).

Configuration

ParamètreDéfautDescription
IP de destination255.255.255.255Broadcast (tous les appareils) ou IP unicast spécifique
Port6454Port UDP standard ArtNet
Universe0Numéro d'univers ArtNet (0–15)
Subnet0Sous-réseau ArtNet (0–15)
Net0Réseau ArtNet (0–127)

Vous pouvez configurer plusieurs sorties ArtNet pour envoyer sur différents univers ou adresses IP simultanément.

Assurez-vous que votre ordinateur et vos nœuds ArtNet sont sur le même sous-réseau (typiquement 2.x.x.x ou 10.x.x.x). Le mode broadcast (255.255.255.255) fonctionne sur tous les réseaux mais génère plus de trafic.
