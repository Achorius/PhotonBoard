1. Introduction

1.1 Qu'est-ce que PhotonBoard ?

PhotonBoard est un logiciel professionnel de commande DMX512 pour Mac et Linux. Il permet de piloter des projecteurs intelligents (lyres, scanners, PARs, barres LED, etc.) via les protocoles ArtNet et USB DMX.

L'application offre une interface moderne et intuitive, pensee pour les eclairagistes, les regisseurs et les techniciens du spectacle. Elle integre un visualiseur 3D en temps reel, un systeme de timeline de type DAW, un moteur d'effets parametriques, le mapping MIDI, et le suivi de poursuite par manette.

3 univers DMX

Pilotez jusqu'a 1536 canaux DMX simultanement via ArtNet ou adaptateur USB.

Visualiseur 3D temps reel

Visualisez vos faisceaux, couleurs et mouvements dans un espace 3D WebGL.

Timeline type DAW

Programmez vos sequences avec des clips sur une timeline multi-piste.

Moteur d'effets

Cercle, sinus, rainbow, strobe et plus. Choisissez la forme d'onde, la vitesse, le spread.

MIDI natif

Mappez chaque fader, bouton et scene a un controleur MIDI (APC, Launch Control, etc.).

Import universel de fixtures

OFL, GDTF, QLC+, Avolites, GrandMA, CSV/TSV — importez depuis n'importe quelle source.

1.3 Installation

macOS

Telechargez le fichier .dmg depuis le site officiel ou la page GitHub.
Ouvrez le .dmg et glissez PhotonBoard dans le dossier Applications.
Au premier lancement, faites clic droit > Ouvrir si macOS bloque l'application non signee.

Linux

Telechargez le fichier .AppImage ou .deb.
Pour AppImage : chmod +x PhotonBoard.AppImage && ./PhotonBoard.AppImage
Pour Debian/Ubuntu : sudo dpkg -i photonboard_*.deb
Pour la sortie USB DMX sur Linux, installez libusb-1.0-0 : sudo apt install libusb-1.0-0

2.1 Barre d'outils (Toolbar)

Barre horizontale en haut de la fenetre, toujours visible. Elle contient :

ElementDescription
Nom du showAffiche le nom du show en cours, centre dans la barre.
GM (Grand Master)Curseur horizontal orange. Controle le niveau general de sortie (0-100%). Affecte tous les univers DMX.
MIDI LearnBouton violet. Active le mode d'apprentissage MIDI. Par defaut, mappe le Grand Master. Lorsqu'actif, il clignote en violet.
LiveBouton vert. Demarre/arrete la lecture de la Timeline. Clignote quand la timeline est en cours.
ClearBouton orange. Efface le Programmeur (valeurs manuelles). Devient actif uniquement quand des valeurs ont ete touchees manuellement. Raccourci : Backspace.
BlinderBouton jaune. Maintenez enfonce pour envoyer un flash blanc sur toutes les fixtures. Relacher = arret.
StrobeBouton cyan. Maintenez enfonce pour activer un effet stroboscope sur les fixtures actives.

BlackoutBouton rouge. Coupe toute sortie DMX. Clignote en rouge quand actif. Raccourci : Esc.
Clic droit sur Live, Blinder, Strobe ou Blackout ouvre le MIDI Learn pour mapper ce bouton specifique a un controleur MIDI.

2.2 Panneau Patch (sidebar gauche)

Sidebar permanente a gauche (largeur fixe 192px). Elle affiche la liste de toutes les fixtures patchees.

Contenu du panneau

  • En-tete "PATCH" avec le nombre de fixtures entre parentheses et un bouton + pour ouvrir l'editeur de Patch.
  • Filtre par univers : 4 boutons — All, U1, U2, U3 — pour filtrer l'affichage par univers.
  • Liste des fixtures : chaque fixture affiche son nom, son adresse (ex: U1.001) et son type. Un clic selectionne la fixture. Cmd/Ctrl + clic pour la multi-selection.
  • Mini faders : lorsqu'une seule fixture est selectionnee, des faders compacts apparaissent en bas, affichant les canaux de la fixture (Dimmer, Pan, Tilt, Red, Green, Blue, etc.) avec un code couleur par type.
La selection dans le panneau Patch est synchronisee avec le visualiseur 3D et le Stage Layout. Selectionner une fixture ici la met en surbrillance partout.

2.4 Barre d'executeurs (Executor Bar)

Barre permanente en bas de l'interface, composee de 8 colonnes avec 4 emplacements chacune (32 slots au total).

Fonctionnalites

  • Colonnes colorees : chaque colonne a une couleur personnalisable. Cliquer sur le point colore dans l'en-tete ouvre une palette de 12 couleurs.
  • Titre de colonne : double-cliquer sur le titre pour le renommer.
  • Scenes : les scenes enregistrees dans l'onglet Scenes apparaissent automatiquement dans les slots. Cliquer sur une scene = GO/STOP.
  • Double-clic sur le nom d'une scene pour la renommer.
  • Drag & drop : deplacez les scenes d'un slot a un autre par glisser-deposer.
  • Modes : chaque cellule a un mode (cliquer sur le badge a droite pour cycler) :
ModeComportement
TOG (Toggle)GO / STOP alternes. Un clic demarre, un deuxieme arrete.
TRIG (Trigger)Joue tant que vous maintenez le bouton/la note MIDI.
FLASHPlein fader (255) tant que vous appuyez.
Clic droit sur une scene dans l'Executor Bar ouvre le MIDI Learn pour cette scene specifique. Le mode de la cellule (TOG/TRIG/FLASH) est automatiquement applique au mapping MIDI.
Lorsque PhotonBoard tourne sur un Raspberry Pi en mode kiosk, la barre d’exécuteurs est aussi présente sur le second écran tactile (Stage Window) et dans le navigateur de tout client distant. Tout déplacement, renommage ou changement de mode est synchronisé en temps réel dans les deux sens (voir chapitre 17).

2.5 Barre de statut (Status Bar)

Barre fine tout en bas de la fenetre. Elle affiche en temps reel :

  • ArtNet OK / Offline : etat de la connexion ArtNet avec le nombre d'univers actifs.
  • Nombre de fixtures : ex. "27 fixtures patched".
  • MIDI : nom du peripherique MIDI connecte ou "No device".
  • Dernier message MIDI : affiche en temps reel le type, canal, numero et valeur du dernier message recu.
  • Command Line : champ de saisie rapide pour des commandes textuelles.
  • Fichier sauvegarde : nom du fichier .pbshow actuel. Cliquer dessus ouvre le dossier dans le Finder.
  • Indicateurs GM/BO : affiche le niveau du Grand Master s'il est inferieur a 100%, et "BLACKOUT" en rouge clignotant si actif.

3. Onglet 3D View 1

Le visualiseur 3D affiche une representation tridimensionnelle de votre scene avec les fixtures positionnees dans l'espace et leurs faisceaux lumineux en temps reel (WebGL/Three.js).

3.1 Barre de controle

RoomActive/desactive l'affichage des murs de la salle.
GridActive/desactive la grille au sol.

ShadowsActive/desactive les ombres portees (gourmand en GPU).

3.2 Navigation 3D

ActionControle
Orbiter (tourner autour)Glisser avec le bouton gauche de la souris
ZoomerMolette de la souris
Panoramique (deplacer)Glisser avec le bouton droit de la souris

3.3 Panneau de proprietes (a droite)

Lorsqu'une fixture est selectionnee, un panneau de proprietes apparait en superposition a droite de la vue 3D :

  • Position X / Y / Z : curseurs pour deplacer la fixture dans l'espace (en metres).
  • Aim (Tilt / Dir) : pour les fixtures statiques, orientez l'angle de projection et la direction.
  • Pan/Tilt Invert : pour les lyres, inversez les axes Pan et Tilt si besoin.
  • Beam : angle d'ouverture du faisceau (2 a 90 degres).
  • Mount : type de montage — Ceiling, Floor, Wall Left, Wall Right, Wall Back.
  • Boutons de placement rapide : "Ceiling", "4m", "Floor", "L Wall", "R Wall", "Back", "Reset".

3.4 Controles DMX rapides

Lorsque des fixtures sont selectionnees, une zone de controle apparait sous la vue 3D :

  • Selection de groupe : boutons pour selectionner rapidement un groupe (ex: "TRAD", "475"), "All" ou "None".
  • Presets couleur : 9 pastilles de couleur (R, G, B, blanc, ambre, cyan, magenta, CTB, noir) pour appliquer une couleur en un clic.
  • Sliders : curseurs horizontaux pour Dimmer, Pan, Tilt et Zoom (n'apparaissent que si la fixture les supporte).

4. Onglet Stage Layout 2

Le Stage Layout offre une vue 2D en plan de votre installation, avec un editeur de placement par glisser-deposer.

4.1 Zone principale : vue de dessus

Les fixtures sont representees par des icones sur un plan vu de dessus. Vous pouvez :

  • Cliquer sur une fixture pour la selectionner.
  • Glisser une fixture pour la deplacer.
  • Les fixtures affichent en temps reel leur couleur DMX actuelle.

4.2 Vue de cote (Side View)

Sous la vue de dessus, une vue laterale montre la position en hauteur des fixtures. Le bouton Side dans le panneau Properties permet de l'afficher ou la masquer.

4.3 Panneau Properties (a droite)

Identique au panneau du 3D View : position, montage, angle, etc. Permet de modifier les proprietes de la fixture selectionnee.

4.4 Barre de controle

Memes controles que le 3D View (Room, Beams, Grid, Snap) avec en plus :

  • Export PDF : genere un plan de scene au format PDF avec le nom du show, les dimensions de la salle, et le placement de chaque fixture avec son adresse DMX et son groupe.

4.5 Controles DMX rapides

Comme dans le 3D View : selection de groupe, presets couleur, sliders Dimmer/Pan/Tilt/Zoom.

5. Onglet Timeline 3

La Timeline est un sequenceur multi-piste de type DAW (Digital Audio Workstation). Elle permet de programmer des enchainements de scenes dans le temps.

5.1 Barre de transport

BoutonFonction
Play / StopDemarre ou arrete la lecture de la timeline.
Compteur de tempsAffiche la position actuelle (minutes:secondes ou timecode).
TC OFF / TC ONActive la reception de timecode MIDI (MTC) pour synchroniser la timeline.
ZoomCurseur pour ajuster le niveau de zoom horizontal.
DurationDuree totale de la timeline (en secondes).

5.2 Sidebar des scenes

A gauche, la liste des scenes disponibles. Chaque scene indique son nom, sa couleur et le nombre de fixtures concernees. Vous pouvez glisser-deposer une scene depuis cette liste vers une piste pour creer un clip.

5.3 Pistes et clips

  • Pistes : la timeline peut comporter plusieurs pistes empilees verticalement. Clic droit sur une piste pour ajouter/supprimer des pistes ou renommer.
  • Clips : chaque clip represente une scene placee a un moment donne avec une duree. Les clips sont colores selon la scene.
  • Redimensionner : tirez les bords gauche/droit d'un clip pour changer sa duree.
  • Deplacer : glissez un clip pour le deplacer dans le temps ou vers une autre piste.
  • Clic droit sur un clip : menu contextuel pour supprimer, dupliquer ou modifier.

5.4 Marqueurs

Les marqueurs sont des reperes temporels affiches sous la regle. Ils permettent de marquer des moments cles (debut chorus, noir, etc.).

  • Ajouter : clic droit sur la regle > "Add Marker".
  • Deplacer : glissez un marqueur sur la regle.
  • Renommer : double-clic dans la sidebar des marqueurs.
  • Couleur : 6 couleurs disponibles (jaune, orange, rouge, vert, bleu, violet).

5.5 Zones

Les zones definissent des regions nommees sur la timeline (ex: "Intro", "Refrain", "Outro"). Elles apparaissent sous les marqueurs.

  • Creer : glissez sur la barre de zones pour definir une region.
  • Renommer : double-clic dans la sidebar des zones.
  • Reordonner : glisser-deposer dans la sidebar.
  • Navigation MIDI : les zones et marqueurs peuvent etre mappes a des notes MIDI pour sauter directement a un point de la timeline.
Le bouton Live dans la Toolbar et le raccourci dans l'Executor Bar controlent la meme timeline. Vous pouvez lancer la timeline depuis n'importe quel onglet.

6. Onglet Effects 4

Le moteur d'effets permet de generer des mouvements, changements de couleur et variations d'intensite parametriques en temps reel.

6.1 Selection des fixtures

La colonne de gauche liste les groupes et les fixtures. Selectionnez les fixtures ou le groupe cible avant de lancer un effet.

  • Les groupes sont affiches avec leur couleur et le nombre de fixtures.
  • Les fixtures sont listees en dessous.
  • Un indicateur de compatibilite montre si la fixture supporte les canaux requis par l'effet.

6.2 Templates d'effets

La zone centrale propose des boutons d'effets pre-configures, organises en categories :

CategorieEffets disponibles
MovementCircle, Figure-8, Line H, Line V, Square, Random, Wave, Spiral
ColorRainbow, Color Fade, RGB Chase
IntensitySinus Dimmer, Strobe, Pulse, Buildup, Blackout Fade
BeamZoom Breathe, Iris Pulse, Focus Sweep
SpatialSpatial Circle, Spatial Line, Spatial Wave

6.3 Parametres d'effet

Apres avoir lance un effet, vous pouvez ajuster en temps reel :

  • Speed (BPM) : vitesse de l'effet.
  • Size (%) : amplitude de l'effet (0-100%).
  • Spread (degres) : decalage entre les fixtures (0-360). Permet de creer des vagues, des arcs, etc.
  • Waveform : forme d'onde — Sine, Triangle, Square, Sawtooth, Random.

6.4 Editeur de courbe

Un editeur de courbe avance permet de dessiner des formes d'onde personnalisees pour un controle total sur l'animation.

Les effets en cours sont automatiquement inclus dans les scenes enregistrees via l'onglet Scenes. Les scenes captureront a la fois les valeurs DMX et les effets actifs.

7. Onglet Scenes 5

L'onglet Scenes permet d'enregistrer, organiser et rejouer des etats lumineux complets.

7.1 Enregistrement d'une scene

Configurez l'etat lumineux souhaite en manipulant les faders (onglet Fixtures), les presets couleur (3D View) ou les effets.
Cliquez sur le bouton REC (rouge) en haut a droite de l'onglet Scenes.
Une nouvelle scene est creee avec les valeurs du Programmeur (valeurs touchees manuellement). Si aucune fixture n'est selectionnee, toutes les fixtures avec des valeurs non-nulles sont capturees.
REC + Clear : apres l'enregistrement, cliquez "Clear" (ou Backspace) pour vider le Programmeur et voir la scene "proprement" depuis le playback.

7.2 Structure d'une scene

Chaque scene contient :

  • Nom : modifiable (double-clic ou fleche d'expansion).
  • Nombre de fixtures : affiche entre parentheses.
  • Steps (cues) : une scene peut contenir plusieurs steps. Expandre la scene pour les voir et les editer.
  • Faders de duree : Fade In et Fade Out reglables pour chaque step.
  • Effets snapshotes : si des effets etaient actifs a l'enregistrement, ils sont sauvegardes avec la scene.

7.3 Controles de scene

BoutonFonction
GO (vert)Lance la scene. Le fader de la scene est monte progressivement.
StopArrete la scene.
FaderCurseur horizontal pour controler l'intensite de la scene (0-100%).
MIDI indicatorBadge violet indiquant un mapping MIDI. Ex: "M:32" = note MIDI #32.
Fleches haut/basNaviguer entre les steps d'une scene multi-step.

7.4 Ajouter un step

Expandre une scene existante, modifier l'etat lumineux, puis cliquer + Add Step. Le nouveau step est ajoute a la suite.

8. Onglet Fixtures 6

L'onglet Fixtures offre un controle detaille de chaque fixture, avec des faders par canal, un color picker et un XY pad.

8.1 Selection d'univers

En haut a gauche, les boutons 1, 2, 3 permettent de choisir l'univers affiche. Le bouton Sheet bascule vers une vue en tableau.

8.2 Groupes

La sidebar de gauche affiche les groupes de fixtures. Cliquez sur un groupe pour selectionner toutes ses fixtures. Les groupes disponibles incluent :

  • Groupes personnalises (TRAD, 475, Barre 3, Barre 4, Barre 5, PAR, Wall...)
  • Boutons All (tout selectionner) et None (deselectionner).

8.3 Vue Faders

Chaque fixture de l'univers selectionne est affichee avec ses faders verticaux. Chaque fader controle un canal DMX :

  • Les faders sont codes par couleur selon le type de canal (orange = dimmer, rouge = rouge, vert = vert, bleu = bleu, violet = pan/tilt, etc.).
  • Le nom du canal est affiche sous chaque fader (D, P, T, R, G, B, W, Z, etc.).
  • Des pastilles de couleur en bas representent la couleur courante de chaque fixture.

8.4 Vue Sheet

Le mode Sheet affiche les fixtures dans un tableau compact avec les valeurs de chaque canal. Ideal pour une vue d'ensemble rapide.

8.5 Color Picker

Lorsqu'une fixture RGB est selectionnee, un selecteur de couleur interactif apparait avec :

  • Roue de couleur : selecteur circulaire teinte/saturation.
  • Barre de luminosite : ajustez la luminosite de la couleur choisie.
  • Presets couleur : raccourcis vers les couleurs courantes.

8.6 XY Pad

Pour les lyres (fixtures avec Pan/Tilt), un pad XY interactif permet de controler le Pan (axe X) et le Tilt (axe Y) en glissant avec la souris.

Clic droit sur un fader ouvre un menu avec l'option MIDI Learn pour mapper ce canal specifique a un controleur MIDI.

9. Onglet Patch 7

L'onglet Patch est le coeur de la configuration DMX. C'est ici que vous ajoutez, supprimez et configurez vos fixtures.

9.1 Tableau de patch

Le tableau principal affiche toutes les fixtures avec les colonnes suivantes :

ColonneDescription
#Numero d'ordre
NameNom de la fixture (editable directement dans le tableau)
TypeType/modele de la fixture (ex: BT-Theatre 200TW, Intimidator Spot 475Z)
ModeMode DMX selectionne (menu deroulant). Chaque mode definit un mapping de canaux different.
AddressAdresse DMX sous la forme Un.xxx (ex: U1.001)
ChNombre de canaux occupes
GroupsGroupes auxquels la fixture appartient
DelBouton de suppression

9.2 Ajouter une fixture

Cliquez + Add Fixture en haut a droite.
Choisissez un type de fixture dans la liste (toutes les definitions importees sont affichees).
Selectionnez le mode DMX, l'univers et l'adresse de depart.
Choisissez le nombre d'exemplaires a ajouter. Les adresses sont auto-incrementees.

9.3 Importer des fixtures

Cliquez Import Fixture pour ouvrir la boite de dialogue d'import. PhotonBoard supporte tous ces formats :

  • OFL JSON — Open Fixture Library (format natif)
  • GDTF — General Device Type Format (norme DIN SPEC 15800)
  • QLC+ (.qxf) — Fichiers de la bibliotheque QLC+
  • Avolites (.d4, .r20) — Personality files Avolites
  • GrandMA (.xml) — Fichiers XML d'export GrandMA
  • CSV / TSV — Tableaux avec colonnes standardisees
Vous pouvez aussi telecharger des fixtures directement depuis la bibliotheque en ligne Open Fixture Library via le panneau de recherche OFL integre a la boite de dialogue "+ Add Fixture".

9.4 Groupes

Le panneau Groups a droite permet d'organiser les fixtures :

  • + Add : creer un nouveau groupe avec nom et couleur.
  • Glisser-deposer : glissez une fixture (ou une selection) depuis le tableau vers un groupe.
  • Sous-groupes : les groupes supportent la hierarchie (glisser un groupe dans un autre).
  • Chaque groupe affiche le nombre de fixtures entre parentheses.

9.5 Fixture Editor

L'editeur de fixture integre permet de creer ou modifier des definitions de fixtures personnalisees. Vous pouvez definir les canaux, les modes, et sauvegarder dans la bibliotheque locale.

10. Onglet MIDI 8

L'onglet MIDI gere les peripheriques MIDI connectes et tous les mappings entre votre controleur et PhotonBoard.

10.1 Peripheriques (colonne gauche)

Inputs

Liste des peripheriques MIDI d'entree detectes, avec indicateur de connexion (vert = connecte, rouge = deconnecte). Bouton Rescan pour re-detecter.

Outputs

Liste des peripheriques de sortie MIDI.

Monitor

Affiche en temps reel le dernier message MIDI recu :

  • Type : CC (Control Change) ou NOTE
  • Channel : canal MIDI (1-16)
  • CC# / Note : numero du controleur ou de la note
  • Value : valeur (0-127)

10.2 Mappings (partie principale)

Tableau listant tous les mappings MIDI actifs :

ColonneDescription
NameNom descriptif du mapping
SourceMessage MIDI source (ex: CC Ch1 #48)
TargetCible dans PhotonBoard (ex: channel > Dimmer, cuelist_go)
ModeComportement : Direct, Toggle, Trigger, Flash
RangePlage de valeurs (0-255 par defaut)
XBouton de suppression du mapping

10.3 Quick Learn

Le bouton Quick Learn (en haut a droite) active le mode MIDI Learn global. Dans ce mode :

Cliquez sur Quick Learn — le bouton devient rouge et clignote.
Bougez un fader ou appuyez sur un bouton de votre controleur MIDI.
Le message est capture et un nouveau mapping est cree automatiquement.

10.4 Modes de comportement

ModeComportement
DirectContinu / proportionnel. La valeur MIDI (0-127) est mappee sur la plage DMX (0-255).
ToggleAppuyer = ON, appuyer a nouveau = OFF.
TriggerON tant que le bouton est maintenu, OFF au relachement.
FlashPleine valeur (255) tant que le bouton est maintenu.
Le moyen le plus rapide de mapper un element est le clic droit directement sur l'element (bouton Toolbar, fader dans Fixtures, scene dans Executor Bar), puis de bouger un controle MIDI.

11. Onglet Follow 9

Le mode Follow transforme un gamepad (manette Xbox, PlayStation, etc.) en controleur de poursuite pour piloter des lyres en temps reel.

11.1 Gamepad

L'indicateur en haut montre l'etat de connexion de la manette. Branchez un gamepad USB ou Bluetooth compatible.

11.2 Selection des fixtures

La section "Follow Fixtures" liste toutes les lyres patchees (fixtures de type Moving Head). Cochez les fixtures a inclure dans la poursuite.

11.3 Controles

ParametreDescription
Trigger buttonBouton de la manette qui active la poursuite. Cliquez LT pour changer (mode learn).
SpeedVitesse de suivi (sensibilite du mouvement).
Target heightHauteur de la cible au sol (en metres).
DimmerNiveau du dimmer des fixtures en poursuite (0-100%).
Invert X / Z / YCases a cocher pour inverser les axes de controle si besoin.

11.4 Utilisation

Selectionnez les lyres dans la liste Follow Fixtures.
Placez-les dans le Stage Layout (indispensable pour le calcul des angles).
Maintenez LT sur la manette pour activer le suivi.
Stick gauche : deplacer la cible en X/Z (position au sol).
Stick droit Y : ajuster la hauteur de la cible.
Relachez LT pour desactiver le suivi.
Les fixtures doivent etre positionnees dans le Stage Layout pour que le calcul des angles Pan/Tilt fonctionne correctement.

12. Onglet Settings 0

La page Settings permet de configurer le show, la sortie DMX et les peripheriques USB.

12.1 Show

  • Show Name : nom du show courant (champ texte editable).

14. Gestion des fichiers Show

14.1 Format de fichier

Les shows sont sauvegardes au format .pbshow (JSON compresse). Chaque fichier contient :

  • Le nom et les metadonnees du show
  • La configuration ArtNet
  • Le patch complet (fixtures, adresses, modes, groupes)
  • Toutes les scenes (cuelists) et leurs steps
  • Les chases
  • Les mappings MIDI
  • La configuration de la salle 3D (roomConfig)
  • La timeline complete (clips, marqueurs, zones, nombre de pistes)

14.2 Sauvegarde automatique

PhotonBoard charge automatiquement le dernier show sauvegarde a chaque lancement. Le chemin du fichier est affiche dans la barre de statut.

14.3 Emplacement par defaut

Les fichiers sont stockes dans le dossier ~/Library/Application Support/PhotonBoard/shows/ (macOS) ou ~/.config/PhotonBoard/shows/ (Linux).

Cliquer sur le nom du fichier dans la barre de statut ouvre le dossier contenant dans le Finder/gestionnaire de fichiers.

16. Sortie DMX : ArtNet & USB

16.1 Architecture de sortie

PhotonBoard genere les valeurs DMX a environ 40 Hz (25 ms par trame). Les valeurs sont calculees par un mixer central qui fusionne les sources :

  1. Programmeur : valeurs manuelles (faders, presets couleur, XY pad) — priorite la plus haute.
  2. Scenes (cuelists) : valeurs enregistrees dans les scenes actives, avec fades.
  3. Effets : valeurs generees par le moteur d'effets.
  4. Timeline : clips actifs de la timeline.

Le resultat est envoye simultanement a toutes les sorties configurees (ArtNet + USB).

16.2 ArtNet

Protocole reseau standard pour le DMX sur IP. PhotonBoard envoie des paquets ArtDMX via UDP.

  • 3 univers simultanement.
  • Mode broadcast (255.255.255.255) par defaut.
  • Compatible avec les nodes ArtNet (ENTTEC ODE, Chauvet DMX-AN, etc.).

16.3 USB DMX

Support des adaptateurs USB-DMX les plus courants :

  • ENTTEC Open DMX USB / DMX USB Pro
  • Eurolite USB-DMX512 / USB-DMX512 PRO
  • Showtec Quick DMX
  • Beamz DMX60 / DMX70
  • Velleman VM116 / K8062
  • Stairville USB-DMX
  • uDMX / Anyma / Clone
  • Generic FTDI Adapter
La sortie USB DMX necessite le module serialport. Sur Linux, assurez-vous que l'utilisateur a les droits d'acces au port serie (sudo usermod -a -G dialout $USER).

17. Raspberry Pi & mode Remote

PhotonBoard peut fonctionner en autonomie sur un Raspberry Pi 5 installé en régie : double écran tactile, sortie ArtNet, contrôle distant depuis n’importe quel ordinateur, tablette ou téléphone du réseau local. Ce chapitre décrit l’installation, le mode Remote, la synchronisation bidirectionnelle, et l’intégration Bitfocus Companion.

17.1 Installation sur Raspberry Pi 5

Une image système prête à l’emploi est fournie dans dist/PhotonBoard-v0.1.0-<date>.img.gz. Elle pèse environ 1.5 Go compressée et tient sur n’importe quelle carte SD à partir de 16 Go. La partition est automatiquement étendue à la taille réelle de la carte au premier démarrage.

Ouvrir Raspberry Pi Imager sur Mac ou PC.
Choisir l’OS → Use custom → sélectionner le fichier .img.gz.
Cliquer sur l’icône d’engrenage pour personnaliser le hostname (ex. : AulaFlorence) et le réseau Wi-Fi de la salle.
Flasher la carte SD, l’insérer dans le Pi 5, brancher les écrans HDMI et le câble réseau, puis alimenter.
Au premier boot l’image se redimensionne et redémarre. Après 30 s environ, PhotonBoard s’affiche en plein écran sur la sortie HDMI 1.

17.1.1 Configuration des écrans tactiles

L’image inclut un script (/usr/local/bin/photonboard-map-touch) qui associe chaque dalle tactile USB wch.cn à la sortie HDMI correspondante en se basant sur le chemin USB physique. Cette association reste stable après un redémarrage, contrairement aux ID xinput qui dépendent de l’ordre d’énumération USB.

Si le côté gauche/droite est inversé, éditer /etc/photonboard/touch-map.conf et passer SWAP=0 à SWAP=1, puis redémarrer.

17.2 Stage Window (deuxième écran tactile)

Lorsque deux écrans HDMI sont détectés, PhotonBoard ouvre automatiquement une fenêtre Stage sur le second écran. Cette surface tactile est optimisée pour le pilotage live :

  • Grille 8 × 4 (32 scènes) reproduisant fidèlement la barre d’exécuteurs de la fenêtre principale.
  • Couleurs de colonnes, badges TOG/TRIG/FLASH et faders identiques à la fenêtre principale.
  • Drag & drop tactile : maintenir une cellule pour la déplacer vers une autre case — la modification se propage instantanément à la fenêtre principale et à tous les clients distants.
  • Boutons globaux Blackout, Blinder, Strobe, et fader Grand Master.
L’onglet 3D View est désactivé sur les Raspberry Pi (GPU ARM trop limité pour la 3D temps réel). Pour prévisualiser en 3D, utilisez le mode Remote depuis un Mac (voir 17.3).

17.3 Mode Remote (Mac, tablette, téléphone)

Tout appareil du réseau local (Mac, iPad, iPhone, Android, Windows…) peut se connecter à PhotonBoard via un simple navigateur web :

Sur l’ordinateur, ouvrez http://PhotonBoard.local:9091
Sur un téléphone connecté au même Wi-Fi, ouvrez la même URL.

17.4 Synchronisation bidirectionnelle des exécuteurs

Toutes les actions sont synchronisées en temps réel dans les deux sens entre le Pi et chaque client distant :

  • Déplacer une scène dans la barre d’exécuteurs (sur le Pi, dans le navigateur ou sur tablette) met à jour instantanément la grille sur tous les autres écrans, avec couleurs et badges de mode préservés.
  • Déclencher une scène, modifier un fader, basculer le Blackout : tout est diffusé à ~4 Hz sur le réseau (assez fluide pour le retour visuel, économe en bande passante).
  • Un guard flag empêche les boucles de rétropropagation : une mise à jour entrante n’est jamais renvoyée à son émetteur.
Si la grille semble ne pas se mettre à jour après un déplacement dans le navigateur, faites un rechargement forcé (Cmd+Shift+R) — le navigateur peut avoir mis en cache une ancienne version de l’interface.

17.5 Bitfocus Companion & API WebSocket

L’image Pi inclut Bitfocus Companion et le module companion-module-photonboard pré-installés. Vous pouvez piloter chaque scène, le Grand Master, le Blackout et la Timeline depuis un Stream Deck ou tout contrôleur compatible Companion.
